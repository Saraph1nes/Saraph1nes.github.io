npm快速换源工具nrm

为什么要切换npm源

1.速度太慢

因为默认的npm源是国外的，速度比较慢。可以选择国内镜像，加快下载安装速度，比如我们可以切换到taobao源或者公司内部的源。

2.手动切换太麻烦

切换源时，往往记不住源链接，百度之后再来执行命令npm config set registry ….

切换npm源推荐使用nrm

nrm 是一个 js 模块，是一个命令行工具，可以用来快速切换 npm 源。
